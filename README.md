# Learning-tracker

# 📚 Programmer Learning Tracker with Interactive Games

## 📖 Overview

The **Programmer Learning Tracker** is a Python-based console application designed to help beginner programmers organize their learning, track daily practice, and improve problem-solving skills through interactive mini-games.

The application combines a **Learning Tracker** with three logic-based games:
- 🎯 Number Guessing
- ❌⭕ Tic-Tac-Toe
- 🧩 Sudoku

It also saves user progress using file handling, allowing users to continue their learning journey even after closing the application.

---

## 🚀 Features

### 📌 Learning Tracker
- Add programming concepts
- Log daily practice
- View tracked concepts
- Save progress to a text file
- Load previous progress automatically

### 🎮 Interactive Games
- Number Guessing Game
- Tic-Tac-Toe
- Sudoku

### 💾 Progress Tracking
- Stores learning concepts
- Records practice logs
- Saves user progress using file handling

## 💾 Data Persistence

The application stores user progress in **tracker.txt** using Python file handling.

Saved information includes:
- Programming concepts
- Practice history
- Learning progress

When the application starts, previously saved data is automatically loaded.
